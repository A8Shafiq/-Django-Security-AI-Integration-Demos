🛡️ Mitigation Techniques for Stored XSS

  • Output Encoding: Never use |safe on user-generated content. Django auto-escapes by default — leave it on!
  • Input Sanitization: Use libraries like bleach to strip dangerous tags while allowing safe HTML (e.g., <b>, <i>)
  • Content Security Policy (CSP): Add Content-Security-Policy headers to prevent inline script execution
  • HTTPOnly Cookies: Set HttpOnly flag on session cookies to prevent JavaScript from accessing them
  • Input Validation: Validate and reject input that contains suspicious patterns (as an additional layer)
