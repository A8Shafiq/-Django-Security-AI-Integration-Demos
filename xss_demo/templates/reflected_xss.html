🛡️ Mitigation Techniques for Reflected XSS

  • Output Encoding: Always use Django's auto-escaping. Never use |safe on user input from URL parameters.
  • Input Validation: Validate and sanitize query parameters on the server side before reflecting them.
  • Content Security Policy: Add CSP headers: Content-Security-Policy: script-src 'self'
  • URL Encoding: Encode special characters in URLs before reflecting them
  • X-XSS-Protection Header: Enable browser's built-in XSS filter (legacy but still useful)
