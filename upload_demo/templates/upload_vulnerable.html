⚠ INTENTIONALLY VULNERABLE

This upload endpoint performs zero validation. Any file type, any size, any filename is accepted and stored on the server exactly as-is. Upload a file below and see how the server processes it — and what an attacker could do with this weakness.

What's Missing (Why This Is Dangerous)

  • No file extension check — .php, .exe, .sh all accepted
  • No MIME-type validation — Content-Type header is ignored entirely
  • No file size limit — attacker can upload gigabytes of data
  • Original filename preserved — enables path traversal attacks
  • No content inspection — magic bytes, embedded scripts not checked
